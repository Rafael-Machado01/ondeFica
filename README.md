# 📦 Projeto ondeFica?

Aplicação web que consulta endereços a partir de um CEP utilizando a API
ViaCEP.
![Acesse](https://onde-fica.vercel.app/)
## ![Demostração do app](ondeFica.gif)

## ✨ Funcionalidades

- 🔎 Consulta de CEP em tempo real
- 🧠 Validação de CEP inválido
- 🚨 Tratamento de erros
- 💎 Interface moderna estilo glass
- 📱 Layout responsivo
- 📚 Código está todo comentado e explicado.

---

## 🛠 Tecnologias Utilizadas

- React
- TypeScript
- API ViaCEP
- CSS / Tailwind
- Componentização
- Hooks personalizados

---

## 🔃 Fluxo da Aplicação

1. Usuário digita o CEP
2. Input valida o CEP
3. Requisição é enviada para a API ViaCep
4. Resultado é exibido na interface
5. Em caso de erro, feedback visual é mostrado

---

## 🌎 API Utilizada

ViaCEP\
https://viacep.com.br

Exemplo de requisição:

https://viacep.com.br/ws/19802360/json

## 🚀 Como rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/seuusuario/projeto-cep

# Entre na pasta
cd projeto-cep

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
